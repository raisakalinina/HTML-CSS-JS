фронтендик

# Учебные проекты на HTML / CSS / JavaScript

В этом репозитории собраны мои небольшие учебные и pet-проекты на JavaScript, созданные в процессе самостоятельного обучения.  
Каждый проект направлен на отработку конкретных концепций языка и работы с DOM.


## Модальное окно
Интерактивное модальное окно с открытием и закрытием по кнопке, клику на оверлей и нажатию клавиши Escape.  
Отработка:
- работы с DOM
- событий (`click`, `keydown`)
- управления CSS-классами

## Игра «Алфавит»
Мини-игра, в которой элементы можно перемещать кликом мыши.  
Цель - выстроить буквы в правильном порядке, после чего проигрывается звук победы.  
Отработка:
- работы с DOM-структурой
- обработки событий
- логики проверки состояния приложения

## Деструктуризация, rest и spread операторы
Учебный проект для практики современных возможностей JavaScript:
- деструктуризация объектов и массивов
- операторы `rest` и `spread`
- работа с вложенными структурами данных

## Работа с классами
Динамическое создание карточек товаров с использованием классов.  
Реализован расчет скидки и рендер элементов на страницу.  
Отработка:
- ES6 классов
- конструкторов
- динамического HTML-рендера

## Set и Map
Учебный проект для изучения структур данных `Set` и `Map`.  
Реализована обработка событий футбольного матча с подсчетами и выводом статистики.  
Отработка:
- работы с коллекциями
- перебора данных
- форматирования вывода

## Банковское приложение
Интерактивное банковское приложение с авторизацией и управлением счетом.  
Функциональность:
- вход в аккаунт
- переводы между счетами
- сортировка операций
- таймер авто-выхода
- форматирование дат и валют

Отработка:
- работы с массивами и объектами
- методов массивов (`map`, `filter`, `reduce`)
- взаимодействия с DOM
- логики состояния приложения

## Методы массивов
Учебный проект для глубокой практики методов массивов:
- `map`, `filter`, `reduce`, `find`, `some`, `sort`
- работа с объектами
- цепочки методов

## Углубленная работа с DOM
Проект с множеством UI-элементов:
- модальные окна
- плавная прокрутка
- табы
- липкое меню
- ленивый рендер изображений
- слайдер

Отработка:
- делегирования событий
- Intersection Observer API
- оптимизации DOM-взаимодействий

## Функции-конструкторы и прототипы
Практика классического прототипного наследования:
- функции-конструкторы
- работа с `prototype`
- переопределение методов

## Классы, геттеры и сеттеры
Проект для изучения:
- геттеров и сеттеров
- инкапсуляции
- преобразования единиц измерения

## Наследование классов
Реализация и сравнение:
- прототипного наследования
- ES6-классов
- приватных полей

## Дневник тренировок
Небольшое SPA-приложение для учета тренировок с использованием карты.  
Функциональность:
- добавление тренировок
- работа с геолокацией
- отображение маркеров на карте
- сохранение данных в LocalStorage

Отработка:
- ООП
- наследования
- работы с внешними библиотеками
- управления состоянием приложения

## Используемые технологии
- HTML
- CSS
- JavaScript (ES6+)
- DOM API
- LocalStorage
